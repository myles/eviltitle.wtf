# eviltitle.wtf

## Development

### Setup

* PHP 5+
* Node.js
* Bower
* Sass for Ruby

Install Grunt and Bower:

```
$ npm install -g grunt-cli bower
```

Install the Node.js and Bower dependencies:

```
$ npm install
$ bower install
```

### Usage

To run the development web server:

```
$ grunt develop
```
